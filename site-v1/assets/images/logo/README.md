# Logo Images

Place logo image files here when available.

## Recommended files:
- `logo-primary.png` — Full colour logo on transparent background (min 400px wide)
- `logo-white.png` — White version for dark backgrounds (min 400px wide)
- `logo-square.png` — Square crop for social media / favicon use
- `favicon.ico` or `favicon.png` — Site favicon (32x32 or 64x64)

## Usage:
Once images are added, update the `.site-logo` HTML in all pages to use an `<img>` tag
instead of the current CSS text logo, or keep the CSS logo and supplement with
an `<img>` version for Open Graph / social sharing meta tags.

## Notes:
- The CSS logo (Montserrat 200 "SARCASTIC" + 900 "SIGNAGE" with gradient) is the
  current implementation. It renders entirely in-browser with no image dependency.
- Brand gradient: Deep Plum #4A1942 → Magenta #B5294B → Red-Orange #D4410F →
  Terracotta #C16535 → Mossy Green #6B7C3E
